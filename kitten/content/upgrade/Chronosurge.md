[Chronosurge]( {{< relref "upgrade/Chronosurge.md" >}}) is a late-game upgrade that automates the generation of **Temporal Flux**, a resource normally gained only by being offline or inactive. It fundamentally changes how you manage time mechanics by allowing you to generate flux while actively playing.

### **Core Function**
*   **Effect:** It grants [Chronospheres]( {{< relref "buildings/Chronosphere.md" >}}) the ability to generate **1 unit of Temporal Flux per year**.
*   **Mechanic:** This generation triggers not only when a game year passes naturally but also **every time you skip a year** by shattering Time Crystals.
*   **Strategic Value:** This passive generation allows you to keep **Tempus Fugit** (a mode that boosts game speed and production by 1.5x) active permanently, provided you have enough Chronospheres and are shattering crystals frequently.

### **Unlock Requirements and Costs**
Chronosurge is a significant investment, particularly regarding the initial Temporal Flux required to unlock it. You typically need to spend time offline (or paused) to accumulate this initial bank.

*   **Prerequisite:** You must have built at least **1 Chronocontrol** (which in turn requires the [Paradox Theory]( {{< relref "techs/Paradox_Theory.md" >}}) technology).
*   **Cost:**
    *   **Temporal Flux:** 6,500
    *   [Void:]( {{< relref "resource/void.md" >}}) 750
    *   [Unobtainium:]( {{< relref "resource/unobtainium.md" >}}) 100,000
    *   [Time Crystals:]( {{< relref "resource/Time_Crystals.md" >}}) 25

### **Synergy: The "Cryo Engine"**
Chronosurge is a critical component of the late-game "Shatter" or "Cryo" engines.
1.  **Mass Chronospheres:** By building a large number of Chronospheres (e.g., 60–70+), you increase the flux generated per year skip.
2.  **Shattering:** When you shatter Time Crystals to skip years, you instantly trigger the flux generation for every year skipped.
3.  **Infinite Flux:** With enough Chronospheres, a single shatter generates more flux than is consumed by **Tempus Fugit**, allowing you to run the game at 1.5x speed indefinitely while stockpiling flux for **Cryochamber** repairs.
